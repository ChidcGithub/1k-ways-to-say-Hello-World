# #0046 · Dataclass-Based Hello

## Description

A Hello World that models the greeting as a `@dataclass` (PEP 557, Python 3.7+). The `Greeting` class declares a single typed field, `text`, and the decorator auto-generates `__init__`, `__repr__`, and `__eq__`. An instance is created with `Greeting("Hello, World!")`, its `.text` is written to `hello.txt`, and the file is then read back and printed to verify the round-trip. Unlike entry #0009 (a hand-written class), the boilerplate is generated by the decorator; unlike entry #0019 (plain `open()`), the greeting lives inside an object whose shape is declared via type annotations.

## Source Code

`main.py`:

```python
"""Model a greeting as a dataclass and persist it to a file."""

from dataclasses import dataclass


@dataclass
class Greeting:
    text: str


def main() -> None:
    greeting = Greeting("Hello, World!")

    with open("hello.txt", "w", encoding="utf-8") as f:
        f.write(greeting.text)

    with open("hello.txt", "r", encoding="utf-8") as f:
        content = f.read()

    print(content)


if __name__ == "__main__":
    main()
```

## Run

```bash
cd hello-worlds/python/0046-dataclass-based
python main.py
```

Requires Python 3.7 or newer. After running, a file `hello.txt` will appear next to `main.py` containing `Hello, World!`, and the same string will be printed to stdout.
